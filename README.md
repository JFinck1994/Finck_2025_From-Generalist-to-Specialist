# Introduction
This repository compiles the finalized R Markdown scripts used for the analyses reported in 'Plant diversity induces shifts from microbial generalists to specialist by enhancing niche 1 differentiation, microbiome connectivity, and network stability in a temperate grassland' (Finck et al., 2025). 

# Data availability
All raw sequencing data files are deposited in the NCBI Sequence Read Archive (SRA) under the identifications PRJNA645591 (16S sequences) and PRJNA1260230 (ITS sequences). 

# Analysis files 
All analysis files (e.g. ASV tables, metadata etc.) are available upon reasonable request through the corresponding author.
